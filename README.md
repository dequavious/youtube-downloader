### STEPS TO RUN WEB APP

1. Create virtual environment
```
$ python3 -m venv venv
```
2. Activate virtual environment
``` 
$ . venv/bin/activate
```
3. Install required packages
```
$ pip install -r requirements.txt
```
4. Run app.py
```
$ python3 wsgi.py
```
5. Open "http://0.0.0.0:5000/" in browser
